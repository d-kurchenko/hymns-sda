<h1>288. Прославим Бога одной семьёю</h1>

Прославим Бога одной семьёю, направим взор свой к небесам.
              Gm      Gm/E      Dm/F           Gm       A7    Cm-D7(Dm)
// Кто любит Господа, кто любит Господа, соединяйтесь, песня одна. //

Когда-то были совсем чужие, теперь для Бога уже свои.
// Кто любит Господа, кто любит Господа, в белых одеждах встретитесь вы! //

Нас утешает святое Слово, Иисус сказал, к тому придёт,
// Кто любит Господа, кто любит Господа, обитель чудную в сердце найдёт//

Проходят годы, стареют люди, но улыбнитесь, звучат слова:
// «Кто любит Господа, кто любит Господа, готовы встретить вас небеса» //
