<h1>83. Бог есть любовь</h1>

<p>1. Хочу я быть послушным<br>
Иисусу моему,<br>
Хочу воспеть о Нём я<br>
Творению всему!</p>

<p><i>Припев:</i><br>
"Бог есть любовь, Бог есть любовь",<br>
Мне вторит птичий хор,<br>
"Бог есть любовь, Бог есть любовь",<br>
Я петь всегда готов!</p>

<p>2. Хочу я быть с Иисусом<br>
Там в светлой вышине,<br>
Где Он в любви безмерной<br>
Готовит место мне.</p>

<p><i>Припев:</i><br>
"Бог есть любовь, Бог есть любовь",<br>
Мне вторит птичий хор,<br>
"Бог есть любовь, Бог есть любовь",<br>
Я петь всегда готов!</p>

<p>3. Споём же песню дружно<br>
О Божией любви,<br>
Ведь это всем нам нужно,<br>
Бог всем любовь дарит.

<i>Припев:</i><br>
"Бог есть любовь, Бог есть любовь",<br>
Мне вторит птичий хор,<br>
"Бог есть любовь, Бог есть любовь",<br>
Я петь всегда готов!<hr></p>
