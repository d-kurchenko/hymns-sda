<h1>289. МЕСТА НЕТ ДЛЯ ИИСУСА</h1>

<p>Места нет для Иисуса,<br>
В сердце - грех и суета,<br>
Сердце занято заботой,<br>
Нет там места для Христа.</p>

<p><i>Припев:</i><br>
Место есть для злых деяний,<br>
Для Иисуса вход закрыт.<br>
Подвиг мук, скорбей, страданий,<br>
Что свершил Христос, забыт.</p>

<p>Время есть для развлечений,<br>
Для бесплодного труда,<br>
Для суетных наслаждений, -<br>
Нет минуты для Христа.</p>

<p><i>Припев:</i><br>
Место есть для злых деяний,<br>
Для Иисуса вход закрыт.<br>
Подвиг мук, скорбей, страданий,<br>
Что свершил Христос, забыт.</p>

<p>Дай же место Иисусу,<br>
Отвори же сердца дверь,<br>
О, впусти Его скорее,<br>
О, впусти Его теперь!</p>

<p><i>Припев:</i><br>
Место есть для злых деяний,<br>
Для Иисуса вход закрыт.<br>
Подвиг мук, скорбей, страданий,<br>
Что свершил Христос, забыт.</p>
