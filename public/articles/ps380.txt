<h1>380. ВЕРНЫХ БОГУ ОЖИДАЕТ</h1>

<p>Верных Богу ожидает<br>
Та отчизна в небесах,<br>
Где нет скорби и болезни,<br>
Где неведом смерти страх.</p>

<p><i>Припев:</i><br>
Там покой для утружденных,<br>
Для всех скорбью отягченных,<br>
После трудных дней печали<br>
Там покой для всех.<br>
Там не будет бурь свирепых,<br>
Будет Бог хранить спасенных;<br>
Средь плодов, цветов Едемских<br>
Верных радость ждет.</p>

<p>Там Господь теперь готовит<br>
Вечную обитель нам,<br>
И когда Он все окончит,<br>
То придет за нами Сам.</p>

<p><i>Припев:</i><br>
Там покой для утружденных,<br>
Для всех скорбью отягченных,<br>
После трудных дней печали<br>
Там покой для всех.<br>
Там не будет бурь свирепых,<br>
Будет Бог хранить спасенных;<br>
Средь плодов, цветов Едемских<br>
Верных радость ждет.</p>

<p>Там разлуки уж не будет,<br>
Грех и смерть удалены,<br>
И победными венками<br>
Будем мы награждены.</p>

<p><i>Припев:</i><br>
Там покой для утружденных,<br>
Для всех скорбью отягченных,<br>
После трудных дней печали<br>
Там покой для всех.<br>
Там не будет бурь свирепых,<br>
Будет Бог хранить спасенных;<br>
Средь плодов, цветов Едемских<br>
Верных радость ждет.</p>

<p>Кто Христа здесь ожидает, -<br>
Возликуйте! - скоро мы<br>
Здесь, на время умирая,<br>
Будем все воскрешены.</p>

<p><i>Припев:</i><br>
Там покой для утружденных,<br>
Для всех скорбью отягченных,<br>
После трудных дней печали<br>
Там покой для всех.<br>
Там не будет бурь свирепых,<br>
Будет Бог хранить спасенных;<br>
Средь плодов, цветов Едемских<br>
Верных радость ждет.</p>

<p>Там лицом к лицу увидим<br>
Мы Христа на облаках,<br>
Хоры ангелов услышим<br>
На Сионских высотах.</p>

<p><i>Припев:</i><br>
Там покой для утружденных,<br>
Для всех скорбью отягченных,<br>
После трудных дней печали<br>
Там покой для всех.<br>
Там не будет бурь свирепых,<br>
Будет Бог хранить спасенных;<br>
Средь плодов, цветов Едемских<br>
Верных радость ждет.</p>
